# avito-test
Тестовое задание для стажёра Backend (осенняя волна 2025)

Проект запускается с помощью 
```shell
docker compose up
```

e2e тесты запускаются с помощью
(запуск тестов остановит и очистит основной контейнер если он запущен)
```shell
make e2e_tests
```
